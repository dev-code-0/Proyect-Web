## 🚀 Pro Tips

### Tip 1: Contexto es rey
Cada skill LEE tu `product-marketing-context.md` primero. Más detalle = mejor output.

### Tip 2: Combina skills
```
Resultado de /customer-research 
→ Input a /copywriting
→ Input a /page-cro
→ Output final: Landing page optimizada
```

### Tip 3: Itera rápido
```
Version 1 (v1) → Test → Feedback
→ Version 2 (v2) → Test → Feedback
→ Version 3 (v3) → Scale
```

### Tip 4: Usa PRPs
```
"Como Marketing Manager, ¿cuál es la mejor estrategia?"
→ Output específico al rol
vs.
"¿Cuál es la mejor estrategia?"
→ Output genérico
```

### Tip 5: Documenta learnings
Guarda cada insight en `.claude/findings.md`:
```markdown
## Finding 1: Parejas son 60% del target
- Source: /customer-research
- Implicación: Messaging debe ser romántico
- Action: A/B test copy romántico vs. divertido
```

## ❓ FAQ

**P: ¿Necesito tener experiencia de marketing?**
R: No. Los skills hacen el análisis por ti. Solo necesitas contexto.

**P: ¿Cuánto tiempo toma obtener resultado?**
R: Research: 5h | Implementation: 10h | Resultados iniciales: 2-3 semanas

**P: ¿Qué si el skill da output genérico?**
R: Refina tu pregunta, or proporciona más contexto en `product-marketing-context.md`

**P: ¿Puedo usar skills sin entender marketing?**
R: Sí, pero entiende los conceptos (CAC, LTV, conversion rate) para mejores decisiones

**P: ¿Qué skill debería usar primero?**
R: Siempre: `/customer-research` → `/competitor-profiling` → otros
